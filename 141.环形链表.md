## 分析

首先我想到的是把链表的节点都放到 hash表中，如果有重复的就返回True.
时间复杂度：O(n)
空间复杂度：O(n)
```python
class Solution:
    def hasCycle(self, head: ListNode) -> bool:
        #哈希表
        seen = set()
        while head:
            if head in seen:
                return True
            else:
                seen.add(head)
        return False
```

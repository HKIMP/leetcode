## 回溯法

```python
class Solution:
    def combine(self, n, k):
        nums = [i for i in range(1, n+1)]
        
        res = []

        def backtrace(cur, index):
            if len(cur) == k:
                res.append(cur[:])
                return 
            for i in range(index, n):
                cur.append(nums[i])
                backtrace(cur, i+1)
                cur.pop()
            if n == 0 or k == 0:
                return res
        backtrace([], 0)
        return res
```

## 分析

### 迭代

时间复杂度:O(N)
空间复杂度:O(N)

```python
class Solution:
    def inorderTravelsal(self, root: TreeNode) -> List[int]:

        cur, stack, res = root, [], []

        while cur or stack:
            while cur:
                stack.append(cur)
                cur = cur.left
            tmp = stack.pop()
            res.append(tmp.val)
            cur = tmp.right
        return res
```

### 递归

时间复杂度:O(N)
空间复杂度:O(N)

```python
class Soltion:
    def inorderTraversal(self, root: TreeNode) -> List[int]:
        res = []

        def dfs(root):
            nonlocal res #其实因为res 为 [] 是可变类型 不加nonlocal也行
            if not root:
                return
            dfs(root.left)
            res.append(root.val)
            dfs(root.right)
        def(root)
        return res
```

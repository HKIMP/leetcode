## 方法一
1. 复制链表到数组列表中
2. 使用双指针法判断是否是回文

```python
class Solution:
    def isPalindrome(self, head):
        vals = []

        cur = head
        while cur:
            vals.append(cur.val)
            cur = cur.next
        return vals == valas[::-1]
```


## 分析二

快慢指针

```python
class Solution:
    def isPalindrome(self, head):
        fast = head
        slow = head

        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next

        fast = None
        while slow:
            tmp = slow.next
            slow.next = fast
            fast = slow
            slow = tmp

        while fast:
            if fast.val != head.val:
                return False
            else:
                fast = fast.next
                head = head.next
```

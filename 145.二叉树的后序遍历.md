## 迭代

### 迭代一（dfs）模拟递归

时间复杂度:O(N) 每个节点都被遍历一遍
空间复杂度:O(N) 迭代过程中的显示栈开销，最坏情况下 树呈现链状 

```python
class Solution:
    def postorderTraversal(self, root: TreeNode) -> List[int]:

        cur, stack, res = root, [], []
        prev = None

        while cur or stack:
            #思路是 从root开始 遍历到最左
            #下图就是遍历到D
            '''
                        A
                       / \
                      B   C
                     / \
                    D   E     
                     \
                      F
            '''            
            while cur:
                stack.append(cur)
                cur = cur.left
            
            #先弹出栈顶元素
            tmp = stack.pop()

            #这个是判断 tmp 有没有右孩子 后序是 左右根
            #如果没有右孩子，相当于上图没有F
            #如果有右孩子，则判断右孩子是不是已经访问过了
            if not tmp.right or tmp.right == prev:
                res.append(tmp.val)
                prev = tmp
                cur = None
            else:
                stack.append(tmp)
                cur = tmp.right
        return res
```

## 迭代二（bfs）

时间复杂度:O(N)
空间复杂度:O(2N)

```python
class Solution:
    def posterderTraversal(self, root: TreeNode) -> List[int]:

        if not root:
            return []
        
        stack, res = [(0, root)], []

        while stack:
            flag, node = stack.pop()
            if not node: continue

            if flag == 0:
                stack.append((1, node))
                stack.append((0, node.right))
                stack.append((0, node.left))
        
        return res
```
### 迭代三

```python
class Solution:
    def postorderTraversal(self, root: TreeNode) -> List[int]:

        if not root:
            return 

        cur, stack, res = root, [], []

        while cur or stack:
            while cur:
                res.append(cur.val)
                stack.append(cur)
                cur = cur.right
            
            tmp = stack.pop()
            cur = tmp.left

        return res[::-1]
```


### 递归

时间复杂度:O(N)
空间复杂度:O(N)

```python
class Solution:
    def posorderTraversal(self, root: TreeNode) -> List[int]:

        res = []

        def dfs(root):
            nonlocal res
            if not root:
                return
            dfs(root.left)
            dfs(root.right)
            res.append(root.val)
        
        dfs(root)
        return res
```

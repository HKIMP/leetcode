## 暴力法

最简单的想法就是直接遍历每个窗口组，找到最大的元素。然后滑动窗口。

时间复杂度：O(Nk)其中 N 为数组中元素个数。
空间复杂度：O(N-k+1)，用于输出数组。一共有(N-K+1)个窗口

```python
class Solution:
    def maxSlidingWindow(self, nums: List[int], k:int) -> List[int]:

        n = len(nums)
        
        if n*k == 0:
            return []

        return [max(nums[i:i+k]) for i in range(n-k+1)]
```
